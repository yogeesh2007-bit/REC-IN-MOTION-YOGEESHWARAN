# Campus in Motion

A premium college portfolio website that captures the identity, energy, and everyday life of campus through original photography and a modern visual storytelling experience.

## Overview

Campus in Motion is a one-page portfolio website created to present our college through real campus moments, architecture, student interactions, study culture, and open spaces.

The project transforms a collection of live-shot campus photos into a polished digital experience with a cinematic landing page, curated sections, and a clean modern UI.

## Theme

**Student Life in Motion**

This theme focuses on how a college is not just a place of buildings, but a living environment shaped by learning, collaboration, creativity, nature, and movement.

## Features

- Cinematic landing page
- Modern one-page portfolio layout
- Curated campus gallery
- Student life storytelling sections
- Architecture and environment highlights
- Responsive and clean user interface

- Original campus photography

The website was designed to feel:

## Workflow

1. Captured original campus photographs during the event.
2. Selected the strongest images for storytelling.
3. Planned the visual structure and landing page.
4. Designed the UI/UX direction in a premium portfolio style.
5. Built the frontend in Replit.
6. Prepared the project for GitHub submission.

## Photo Story

- Learning spaces

- Campus architecture

- Identity and branding

## Objective

To create a visually strong digital portfolio that represents the spirit of the college through original photography and a polished frontend experience.
